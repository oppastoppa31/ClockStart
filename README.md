
# ClockStart (Add logo here)

This is an Arduino project to automatically start a X model clock timer at 7:50 AM and 11:50 AM, and to reset it to the default clock mode at 2:20 PM.

## Connections

DS3231 SDA -> `20`

DS3231 SCL -> `21`

DS3231 VCC -> `+3.3V`

DS3231 GND -> `GND`

IR G -> `GND`

IR S -> `9`

IR V -> `+3.3V`
